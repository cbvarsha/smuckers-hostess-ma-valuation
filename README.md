# Smucker’s Acquisition of Hostess Brands (TWNK)

## Overview
This project evaluates The J.M. Smucker Company’s $5.6 billion acquisition of
Hostess Brands through a detailed valuation and strategic analysis.

The objective is to assess whether the transaction is financially fair,
value-accretive, and strategically justified from the acquirer’s perspective.

## Deal Summary
- Acquirer: The J.M. Smucker Company
- Target: Hostess Brands Inc. (TWNK)
- Transaction Value: $5.6B
- Offer Price: $34.25 per share (21.9% premium)
- Completion Date: November 2023

## Valuation Methodology
- Discounted Cash Flow (DCF) valuation
- Free Cash Flow to the Firm (FCFF)
- Terminal Value using Gordon Growth Model
- WACC derived using CAPM
- Scenario analysis (Conservative, Base, Optimistic)

## Key Valuation Results
- Base Case DCF Value: $35.76 per share
- Offer Price: $34.25 per share
- Net Present Value (NPV): +$171.5M
- Expected Cost Synergies: ~$100M annually
- Shareholder Value at Risk (SVAR): ~0.86%

## Strategic Rationale
- Expansion into high-growth snacking category
- Revenue diversification away from legacy segments
- Distribution and supply chain synergies
- Margin accretion and EPS accretion potential

## Financing Structure
- Cash and stock transaction
- Approx. 17% debt and 83% equity (market value basis)
- Balanced capital structure preserving financial flexibility
